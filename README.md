# 📱 Rick & Morty - Mobile PWA Wiki

Este projeto é uma enciclopédia móvel baseada na série Rick and Morty, desenvolvida como um **Progressive Web App (PWA)** de alta performance. O objetivo principal foi explorar a integração entre hardware móvel e consumo de APIs assíncronas.

### 🚀 Principais Funcionalidades

* **Consumo de API:** Busca dinâmica utilizando `fetch` e `async/await` na API oficial do Rick and Morty.
* **Busca por Voz:** Integração com a **Web Speech API** para permitir pesquisas sem digitação.
* **Feedback Háptico:** O aplicativo utiliza o motor de vibração do dispositivo (**Vibration API**) para confirmar o sucesso ou erro nas buscas.
* **Modo Offline:** Implementação de **Service Worker** e `manifest.json` permitindo a instalação do app e funcionamento sem internet.
* **UI Mobile-First:** Design moderno com foco em usabilidade móvel e carregamento otimizado.

---

### 🛠️ Tecnologias Utilizadas

* **Linguagens:** HTML5, CSS3 (Modern Grid/Flexbox), JavaScript (ES6+).
* **Hardware:** Web Speech API, Vibration API.
* **DevOps:** GitHub, Netlify (Continuous Deployment).
